# Flutter Tips & Examples

## Repository Overview
This repository contains various Dart files, each showcasing a unique functionality or implementation in Dart. Here is a brief overview of each file:

### Files and Descriptions:

1. **01_debounce_and_form_validation.dart**: An implementation showcasing debounce techniques and form validation in Dart.  
2. **02_interactive_drawing_canvas.dart**: A file that displays how to create an interactive drawing canvas.  
3. **03_bar_charts.dart**: Demonstrates how to create of bar charts with Flutter.
4. **04_copy_and_paste.dart**: Highlights the copying and pasting functionalities, and show different approaches on how to do that in Dart/Flutter.
5. **05_lottie_animations_with_color_filter.dart**: A file dedicated to Lottie animations, specifically, it display how to change the color of a Lottie animation file using the ColorFiltered widget.
6. **06_carousel_slider**: An implementation of a carousel with the carousel_slider library that highlights how to add a functional dot indication using the dots_indicator library. 
7. **07_avatar_stack_animation.dart**: It demonstrates how to use the flutter_animate library to create a smooth animation that overlays multiple avatars on top of each other. 
8. **08_text_and_image_animation.dart**: Another example about how to use the flutter_animate library to animate a text and carousel of images.
9. **09_speech_to_text_with_whisper.dart**: A unique implementation that displays how to use OpenAI's Whisper model to convert speech to text.
10. **10_popover.dart**: This file showcase the popover library, which comes in handy if you want to declutter the UI of your Flutter application.
