# Flutter Tips & Examples

## Repository Overview
This repository contains various Dart files, each showcasing a unique functionality or implementation in Dart. Here is a brief overview of each file:

### Files and Descriptions:

1. **[01_debounce_and_form_validation.dart](./lib/01_debounce_and_form_validation.dart)**: An implementation showcasing debounce techniques and form validation in Dart.  
5. **[02_interactive_drawing_canvas.dart](./lib/02_interactive_drawing_canvas.dart)**: A file that displays how to create an interactive drawing canvas.  
6. **[03_bar_charts.dart](./lib/03_bar_charts.dart)**: Demonstrates how to create of bar charts with Flutter.
7. **[04_copy_and_paste.dart](./lib/04_copy_and_paste.dart)**: Highlights the copying and pasting functionalities, and show different approaches on how to do that in Dart/Flutter.
8. **[05_lottie_animations_with_color_filter.dart](./lib/05_lottie_animations_with_color_filter.dart)**: A file dedicated to Lottie animations, specifically, it display how to change the color of a Lottie animation file using the ColorFiltered widget.
9. **[06_carousel_slider.dart](./lib/06_carousel_slider.dart)**: An implementation of a carousel with the carousel_slider library that highlights how to add a functional dot indication using the dots_indicator library. 
10. **[07_avatar_stack_animation.dart](./lib/07_avatar_stack_animation.dart)**: It demonstrates how to use the flutter_animate library to create a smooth animation that overlays multiple avatars on top of each other. 
11. **[08_text_and_image_animation.dart](./lib/08_text_and_image_animation.dart)**: Another example about how to use the flutter_animate library to animate a text and carousel of images.
12. **[09_speech_to_text_with_whisper.dart](./lib/09_speech_to_text_with_whisper.dart)**: A unique implementation that displays how to use OpenAI's Whisper model to convert speech to text.
13. **[10_popover.dart](./lib/10_popover.dart)**: This file showcase the popover library, which comes in handy if you want to declutter the UI of your Flutter application.
